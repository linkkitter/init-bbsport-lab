# init-bbsport-lab

这是一个用于归档和发布多个独立 HTML 页面的仓库。仓库中的页面以静态 HTML 文件形式存在，不针对任何特定域名或网站，主要用于技术演示、功能测试或内容存档用途。

## 项目简介

`init-bbsport-lab` 是一个轻量级的 HTML 页面集合仓库。所有页面均为独立文件，无需后端服务或数据库支持，可直接通过浏览器打开或部署到静态托管服务进行访问。

本仓库适合存放以下类型的页面：
- 技术实验页面
- 功能原型
- 内容存档
- 临时展示页面

## 目录说明

```
/
├── index.html          # 仓库主页（可选）
├── pages/              # 存放独立 HTML 页面
│   ├── example1.html
│   ├── example2.html
│   └── ...
├── assets/             # 公共资源文件（CSS、JS、图片等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `pages/`：主要页面存放目录，每个 HTML 文件为一个独立页面。
- `assets/`：页面引用的公共资源，按类型分目录存放。

## 使用方式

1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/your-username/init-bbsport-lab.git
   ```
2. 直接双击 `pages/` 目录下的 HTML 文件即可在浏览器中查看。
3. 也可将整个仓库部署到 GitHub Pages 或其他静态托管平台。
